Express Auth Cookies & Sessions

A Node.js and Express.js backend project demonstrating authentication using cookies and sessions.

This project is built for learning and practicing backend authentication concepts using Express.js. It shows how cookies and sessions can be used to manage user login states and preferences.

Features
User login using session and cookie authentication
Remember-me login functionality (extended cookie lifespan)
Protected routes (dashboard access only when logged in)
Logout functionality (session destroy / cookie clear)
Theme preference using cookies (light/dark)
Language preference using cookies (multi-language support)

Tech Stack
Node.js
Express.js
express-session
cookie-parser
